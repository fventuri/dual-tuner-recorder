# SDRplay RSPduo dual tuner recorder utility

`dual-tuner-recorder` is a command line utility to record to disk the stream of I/Q samples from both tuners of the SDRplay RSPduo.

**IMPORTANT NOTICE: this repository is now in read-only mode since this program has been superseded by the newer rsp-recorder utility**

rsp-recorder is available here: https://github.com/fventuri/rsp-recorder
